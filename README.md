# drumtab
Drum tab editor in HTML/JavaScript.
Create tab notation for a drum pattern and then export to a high-resolution image.
Length and grouping of steps is configurable.
Drum parts can be disabled if desired.

Try it here:
https://woodypianoshack.github.io/drumtab/
